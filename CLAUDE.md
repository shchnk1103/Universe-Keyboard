# CLAUDE.md

This file is kept as a compatibility entry point for Claude-style project loading.

The canonical AI workflow and project context now live in:

1. `AGENTS.md` — repository-level collaboration rules.
2. `CONTEXT_INDEX.md` — task-based context routing.
3. `docs/PROJECT_CONTEXT.md` — durable project architecture, build, and design context.
4. `docs/AI_WORKFLOW.md` — multi-agent workflow and handoff rules.

Do not add project history, recent changes, or new architecture notes here. Put durable project context in `docs/PROJECT_CONTEXT.md`, historical changes in `CHANGELOG.md`, and workflow rules in `AGENTS.md` or `docs/AI_WORKFLOW.md`.
